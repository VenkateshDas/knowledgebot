# Telegram Exp

A Telegram bot experiments project managed with [uv](https://github.com/astral-sh/uv).

## Setup

1. Install `uv` if you haven't already.
2. Run `uv sync` to install dependencies and create a virtual environment.
3. Create a `.env` file with your `TELEGRAM_BOT_TOKEN`.

## Usage

Run the bot with:
```bash
uv run telegram_bot.py
```
